# **机器翻译：统计建模与深度学习方法**
*肖桐*  *朱靖波*
*东北大学自然语言处理实验室* $\cdot$ *小牛翻译*

这是一个机器翻译的教程，对统计机器翻译和神经机器翻译进行了全面的介绍。本书可以供计算机相关专业高年级本科生及研究生学习之用，也可以作为自然语言处理，特别是机器翻译相关研究人员的参考资料。本书的所有源代码（latex）均已开放。

本书共分为七个章节，章节的顺序参考了机器翻译技术发展的时间脉络，同时兼顾了机器翻译知识体系的内在逻辑。各章节的主要关系如下：
